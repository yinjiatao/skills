# 产品推广 / 种草文案

## 定位

产品推广文案（种草文案）是一种以激发购买欲望为核心的内容形式，通过生动、真实、有感染力的描述，让读者产生"我也需要这个"的心理共鸣。

目标受众包括：
- **潜在购买者**：正在考虑购买某类产品但尚未决定具体品牌
- **冲动消费者**：容易被情感触发和限时优惠驱动的群体
- **忠实粉丝**：品牌现有客户，可能成为口碑传播者
- **KOL/KOC受众**：通过意见领袖推荐接触品牌的用户

## 写作目标

读者读完后的预期行为/认知改变：

| 阶段 | 预期行为 | 心理变化 |
|------|----------|----------|
| 认知 | 了解产品存在和基本功能 | "原来还有这种产品" |
| 兴趣 | 产生进一步了解的兴趣 | "这个产品看起来不错" |
| 欲望 | 产生拥有的渴望 | "我也想要一个" |
| 行动 | 点击购买链接或前往店铺 | "现在就买" |

核心转化目标：
- 激发情感共鸣和购买欲望
- 降低购买决策的心理门槛
- 建立产品独特价值认知
- 促成即时购买或收藏加购行为

## 核心特征

### 建议字数范围

| 平台类型 | 建议字数 | 说明 |
|----------|----------|------|
| 小红书种草 | 300-800字 | 移动端阅读，需配合图片 |
| 微博/朋友圈 | 100-300字 | 简短有力，适合快速浏览 |
| 电商详情页 | 500-2,000字 | 需要详细展示卖点和使用场景 |
| 短视频脚本 | 100-300字（约30-60秒） | 口语化，强调画面感 |

### 典型结构

种草文案通常遵循**AIDA**或**ACCA**模型：

```
AIDA模型：
Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）

ACCA模型：
Awareness（认知）→ Comprehension（理解）→ Conviction（确信）→ Action（行动）
```

### 语气风格

- **亲切自然**：像朋友推荐，而非推销员推销
- **情感驱动**：强调感受、体验和情绪价值
- **场景化**：将产品融入具体生活场景
- **真实性**：分享真实使用体验，包括小缺点更显可信

## 标准结构模板

### 小红书种草文案模板

```
1. 开场吸引（1-2句话，30-50字）
   - 设置场景或提出问题
   - 示例："秋冬季节，干燥的天气让我们的双手变得粗糙"
   - 使用emoji增加视觉吸引力 ✨

2. 产品介绍（2-3句话，80-120字）
   - 简要介绍产品名称和核心功能
   - 突出1-2个独特卖点
   - 示例："这款护手霜含有天然植物精华，滋养肌肤，修复干燥裂纹"

3. 使用体验（3-5句话，150-250字）
   - 详细描述使用过程和感受
   - 强调感官体验（视觉、触觉、嗅觉等）
   - 对比使用前后的变化
   - 示例："只需抹一次，就能感受到双手变得柔细腻，仿佛喝饱了水一样"

4. 真实反馈（2-3句话，80-150字）
   - 使用时间跨度（如"用了一个月"）
   - 具体变化描述
   - 可加入朋友或家人的反馈增加可信度

5. 购买引导（1-2句话，50-80字）
   - 呼吁行动
   - 可加入限时优惠信息
   - 使用emoji和行动动词
   - 示例："赶快入手这款护手霜，让我们的双手在秋冬季节也能保持柔嫩动人！💕"
```

### 电商详情页文案模板

```
1. 主标题（10-20字）
   - 核心卖点+产品名称
   - 示例："一抹水润｜秋冬必备滋润护手霜"

2. 副标题/卖点提炼（30-50字）
   - 3-4个核心卖点图标化展示
   - 天然成分｜长效保湿｜快速吸收｜不粘腻

3. 痛点共鸣（100-150字）
   - 描述目标用户面临的困扰
   - 建立情感连接
   - "每到秋冬，手部干燥起皮、倒刺横生，让人苦恼..."

4. 解决方案（200-300字）
   - 产品如何解决问题
   - 核心成分解析
   - 技术原理简要说明

5. 使用场景（150-200字）
   - 多场景应用展示
   - 办公室桌面｜床头必备｜随身小包
   - 配合场景图片

6. 用户证言（100-150字）
   - 真实用户评价摘录
   - 使用前后对比

7. 购买保障（80-100字）
   - 售后承诺
   - 退换货政策
   - 限时优惠信息

8. 结尾CTA（30-50字）
   - 强调行动
   - "立即抢购" / "加入购物车"
```

### 6+1文案公式模板

适用于需要全面覆盖产品信息的场景：

```
1. Who（目标受众）
   "适合经常洗手导致手部干燥的上班族"

2. What（产品是什么）
   "XX品牌滋润修复护手霜"

3. When（使用时机）
   "每次洗手后、睡前、出门前"

4. Where（使用场景）
   "办公室、家中、随身携带"

5. Why（购买理由）
   "深层滋养，24小时长效保湿，不油腻"

6. How（使用方法）
   "取适量涂抹于手部，轻轻按摩至吸收"

+1. What If（异议处理）
   "担心过敏？我们提供小样试用，不满意全额退款"
```

## 评级检查表

| 评级 | 要点 |
|------|------|
| Ⅴ | **痛点精准**：准确把握目标用户的具体痛点和需求；解决方案清晰，产品卖点与痛点精准匹配；有明确的情感触发点 |
| Ⅳ | **体验生动**：使用感官化语言描述体验；场景化呈现让用户能够想象使用画面；包含具体的使用效果描述 |
| Ⅲ | **信任建立**：包含真实使用体验（包括小缺点）；有具体的时间跨度和效果变化；避免过度夸大和虚假宣传 |
| Ⅱ | **推销感强**：过度使用营销术语；只讲功能不讲感受；缺乏真实体验描述；通篇都是产品宣传没有价值内容 |
| Ⅰ | **严重错误**：虚假宣传和夸大效果；与实际产品严重不符；抄袭他人内容；诱导性误导信息 |

## 关键写作技巧

### 转化功能为利益（Feature → Benefit）

| 功能(Feature) | 利益(Benefit) | 深层利益(Juicy Benefit) |
|---------------|---------------|------------------------|
| 内置研磨器 | 现磨新鲜咖啡豆 | 每天早上在家享受完美咖啡，不用出门、不用换睡衣、不用花8美元买贵价拿铁 |
| 防水材质 | 保持干爽舒适 | 无论什么天气都保持干爽舒适，不再担心突如其来的雨天 |
| 大容量电池 | 超长续航 | 一周只需充一次电，出行再也不用担心没电 |

### 心理学触发器应用

1. **稀缺性（Scarcity）**："仅剩5件"、"限时48小时"
2. **紧迫性（Urgency）**："今日截止"、"最后机会"
3. **社会认同（Social Proof）**："已有10,000+用户选择"
4. **锚定效应（Anchoring）**：先展示高价再展示实际价格
5. **损失厌恶（Loss Aversion）**："错过再等一年"
