# 社交媒体内容 - 内容分类

## 定位

发布于社交平台的短内容。当前支持：微信朋友圈。
以即时性、情感共鸣、传播性为核心。

## 写作目标

在极短时间内建立共鸣或引发互动（点赞、评论、转发）。

## 核心特征

- **纯文本输出**：无 Markdown 标记，无标题，无加粗，无列表符号
- **短**：通常 50～200 字，不超过 300 字
- **情感浓度高于信息密度**：读者感受 > 读者知道
- **口语感**：读起来像人说的话，而非写出来的文章

## 标准结构模板

### 朋友圈短文案结构

```
开篇（1-2句）→ 核心场景/情绪（3-5句）→ 余韵/互动引导（1-2句）
```

| 部分 | 要点 | 示例 |
|------|------|------|
| **开篇** | 直接入题，无铺垫 | "今天被甲方气笑了" |
| **核心** | 一个场景写透，不贪多 | 具体描述事件 + 感受 |
| **结尾** | 留问句或感叹，引互动 | "你们遇到过吗？" |

### 小红书短笔记结构

```
痛点/好奇引入（1句）→ 个人经历/观察（2-3句）→ 结论/金句（1句）
```

| 部分 | 要点 |
|------|------|
| **引入** | 用"我发现""有没有人"等口语化开头 |
| **经历** | 真实细节 > 华丽辞藻 |
| **结论** | 可反转、可共鸣、可行动 |

## 常见类型

| 类型 | 写法要点 |
|------|---------|
| 情绪抒发 | 具体场景 + 感受，余味结尾 |
| 生活记录 | 细节 + 感受，不堆砌修辞 |
| 观点输出 | 简洁论断，可留问号引互动 |
| 产品种草 | 结合真实场景，不像广告 |
| 活动预热 | 简洁说明利益点，行动号召 |

## 排版规则（⛔ 硬约束延伸）

- 不使用 `#`、`**`、`-`、`>` 等任何 Markdown 符号
- 换行靠自然段落，不靠格式符号
- 可使用 emoji，但不超过 3 个
- 话题标签（#话题#）用于有传播需求时，放结尾
