# 微信（公众号 / 图文号 / 朋友圈）

微信平台包含三种场景，写作规则有差异。根据实际场景加载对应规则。

---

## 场景 A：公众号

### 平台特性

- 受众：25～45 岁中国用户，移动端阅读为主
- 阅读场景：订阅推送，主动点击，注意力较集中
- 传播机制：转发到朋友圈 / 微信群；2025 年起算法推荐（"看一看"）已成主要流量来源，不再依赖纯订阅分发

### 字数规范

| 类型 | 建议字数 |
|------|---------|
| 最低可读 | 600 字 |
| 最佳区间 | 1500～3000 字 |
| 上限 | 5000 字（深度内容）|

### 排版规范

| 评级 | 规则 |
|------|------|
| Ⅴ | 首屏前 3 行必须有吸引力，决定是否继续阅读 |
| Ⅴ | 每段 3～5 行，适合移动端 |
| Ⅴ | 段落间空行，保持视觉呼吸感 |
| Ⅳ | 1500 字以上使用小标题分层 |
| Ⅳ | 关键金句可加粗，每段不超过 1 处 |
| Ⅱ | 超过 8 行的大段落 |
| Ⅰ | Emoji 密集出现（正式内容） |

### 标题写法

| 评级 | 规则 |
|------|------|
| Ⅴ | 设悬念 / 提问题 / 反常识 |
| Ⅴ | 含明确读者受益点 |
| Ⅴ | 兼顾算法：标题含用户搜索关键词，有助于"搜一搜"收录 |
| Ⅰ | 超过 20 字（显示截断） |
| Ⅰ | 纯标题党（平台有降权机制）|

### 内容规则

- 不得诱导分享、诱导关注（"转发即得""关注有福利"受限）
- 涉及医疗、金融、时政内容须特别谨慎
- 外链跳转竞品平台须经微信审核
- AI 批量生成内容会被平台识别并限制推荐；用户停留时长、点赞、分享等行为数据决定算法分发量级

## 场景 B：图文号（"小绿书"）

### 平台特性

- 本质：公众号的"图片/文字"发布模式（非独立平台），因排版酷似小红书，被称为"小绿书"
- 受众：微信全量用户，以"看一看""搜一搜"为主要发现路径；冷启动门槛低，新账号也可获得算法推荐流量
- 阅读场景：碎片化浏览，图片是视觉中心，文字辅助说明
- 传播机制：算法推荐为主，转发朋友圈显示"大图+标题"卡片，比普通文章更抓眼球

### 格式规范

- 图片：最多 9 张（手机端）；订阅号助手支持最多 20 张；图片比例建议 3:4（1080×1440px）
- 文字：最多 1000 字，建议 150～500 字；图片是主角，文字点到即止
- 布局：固定为图片在上（横向滑动）、文字在下，不支持图文混排
- 支持插入：文章链接、小程序、返佣商品（100 粉解锁）
- 图片可嵌入二维码 / 小程序码，支持长按识别，适合私域引流
- 不占用公众号常规群发名额，发布次数不限

### 排版规范

| 评级 | 规则 |
|------|------|
| Ⅴ | 首张图决定点击率，必须清晰、高颜值、主题明确 |
| Ⅴ | 图片质量优先：模糊、光线差的图片在九宫格展示中极为明显 |
| Ⅴ | 文字开头直接切入痛点或价值，无铺垫空间 |
| Ⅳ | 每张图信息独立、自洽，方便用户截图传播 |
| Ⅳ | 可加话题标签（如 #职场成长），有助于搜索收录 |
| Ⅱ | 文字超过 500 字（图文节奏变重） |
| Ⅰ | 图片模糊或素材感强（用户预期是真实内容） |

### 标题写法

| 评级 | 规则 |
|------|------|
| Ⅴ | 痛点+解决方案（如"30岁转行｜我如何靠写作月入2万"） |
| Ⅴ | 数字+场景（如"7天减脂食谱｜跟着吃瘦了5斤"） |
| Ⅴ | 悬念+情绪（如"把生活过成电影，我用了这3个方法"） |
| Ⅰ | 照搬公众号长文标题风格（节奏不匹配） |

### 内容规则

- 平台目前大力扶持，算法推荐流量充足，适合冷启动
- 无原创标记机制（截至 2025 年），但建议避免直接搬运以防风险
- 适合垂类赛道：装修家居、亲子教育、养生保健、情感、职场成长、美食探店等

## 场景 C：朋友圈

### 平台特性

- 受众：个人社交关系链，熟人为主
- 阅读场景：刷朋友圈，极碎片，3 秒决定是否展开
- 传播机制：点赞 / 评论 / 转发

### 字数规范

| 类型 | 建议字数 |
|------|---------|
| 最佳 | 50～150 字 |
| 可接受 | 150～300 字 |
| 避免 | 超过 300 字（需点击"全文"，大多不点）|

### 排版规范

- **纯文本**：不使用任何 Markdown 标记
- 自然换行分段，段落短而有力
- Emoji 可用，不超过 3 个
- 话题标签放结尾，有传播需求时才加

### 内容规则

- 第一句即核心，无铺垫空间
- 真实感优先于精致感
- 商业内容须有真实场景包裹，不能像广告
