# 间隔重复法

## 核心原理

遗忘曲线表明：若无巩固，记忆留存随时间陡降。间隔重复不是考前突击，而是在遗忘即将发生前安排短暂复习，使每次回忆略费力，从而强化长时记忆痕迹。成功回忆后逐步拉长间隔，兼顾效率与巩固；若失败则缩短间隔，避免在虚假掌握上浪费日程。数字化工具可用算法估算个人遗忘点；纸质也可用日历或简单表格实现「先密后疏」的节奏。此法对事实性知识、外语词汇、法条、接口细节等需精确提取的内容尤为有效。深层理解仍需配合应用题、项目与论述输出，否则易出现「记得卡片却不会用」。复习没有节奏、只会重复读笔记时，用间隔把「重复」变成「在正确时间提取」。

## 使用步骤

1. **拆小单元**：将知识拆成可检验的一问一答、一词一义、一概念一辨，避免一张卡塞整章。
2. **初记与首复**：首次学习后，在当天与次日安排第一次主动回忆，再按工具建议扩展间隔（如 3、7、14 天量级）。
3. **先忆后对**：复习时先闭卷回忆再核对答案；失败则标记并重排更近的复习，忌立刻重读代替回忆。
4. **防提示泄漏**：检查题干是否无意中泄露答案，合并冗余相似卡，保持提示与目标匹配。
5. **控总量节奏**：每周审视新增卡量与积压，防止日程爆炸导致放弃；定期退休已内化的卡。
6. **理解型卡片**：对概念加情境题、辨析题或「用一句话解释给外行」类生成任务，避免只剩机械再认。

## 使用场景与案例

### 场景一：外语单词与搭配

**背景**：词表背了忘，阅读时识别慢。

**应用**：正面例句或语境，背面释义与搭配；按软件调度复习，记录易混词群。

**结果**：识别与产出更稳，减少「认得形、不会用」的断层。

### 场景二：医学或法律资格考试

**背景**：条目海量，短期 cram 易崩。

**应用**：高频考点制卡，错题单独成组并加密复习；大节前压测混合卷。

**结果**：长期记忆覆盖考纲核心，临考焦虑下降。

### 场景三：编程语言 API

**背景**：常用函数参数易混，编码时频繁查文档打断心流。

**应用**：卡片正面写需求或错误现象，背面写函数签名、关键参数与坑。

**结果**：常见路径内化，复杂场景仍查文档但次数显著减少。

### 场景四：历史年表与人物

**背景**：年代事件易串，论述缺时间轴。

**应用**：一卡一事，附因果链提示词；Review 时口头串线。

**结果**：大题能按时间轴组织论据，减少张冠李戴。

### 场景五：团队 onboarding

**背景**：新人信息过载，重复提问多。

**应用**：内部术语、流程检查点、合规红线制卡，首月按日轻量推送。

**结果**：提问重复率下降，上手周期缩短，知识有版本可更新。

## 常见误区

- 只做再认题（选择题），缺少主动产出式回忆与简述。
- 间隔过密变成机械重复，过稀则每次像重学，效率双输。
- 卡片措辞含糊，复习时无法判断对错标准与适用范围。
- 只堆新卡不清理已内化内容，日程与心理负担爆炸。
- 忽视睡眠、情绪与情境练习，把算法当万能药。

## 搭配工具

- **康奈尔笔记法**：左栏问题直接导入为卡片正面，总结栏写易错归因。
- **SQ3R 阅读法**：Review 环节与卡片复习合并，复述与提取一致。
- **交叉学习法**：同日卡片混合不同章节与题型，强化判别与迁移。
- **输出倒逼输入法**：周回顾用短文串联本周卡片主题，检验「记得会用」。
