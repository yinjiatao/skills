# 康奈尔笔记法

## 核心原理

康奈尔笔记将一页分为三栏：右侧主笔记区记录课堂或阅读中的实质内容；左侧狭长的线索栏用于课后提炼问题、关键词与提示语，承担提取练习的「钩子」；底部总结栏用少量文字概括本页核心思想、行动项与仍存疑点。其认知价值在于分离「捕捉」与「加工」：当场以跟上节奏为主，课后在线索栏迫使大脑对信息提问式重组，相当于自建题库；总结栏训练压缩与上位概念生成，对抗「记了很多却说不出一句重点」。复习路径被固定为「遮主栏—看线索—回忆—对照」，契合主动回忆与精细加工，使同一页纸在多次复习中产生递增收益。笔记杂乱无章、只会重读不会自测时，三栏结构提供低成本的习惯脚手架。

## 使用步骤

1. **划分版式**：将纸面分为约 2.5cm 左栏、底部约 5cm 总结横条，其余为右栏主笔记区（或使用模板/软件分区）。
2. **主栏速记**：听讲或精读时在右栏用短句、符号快速记录，先求完整捕捉，不求排版完美。
3. **线索转化**：课后 24 小时内在左栏写与右栏对应的问题或提示词，覆盖定义、步骤、对比、例外等类型。
4. **底部总结**：在总结栏写 2～4 句话：本页主题、关键结论、仍疑点或下一步动作。
5. **主动回忆**：复习时遮住右栏，只看左栏自问自答，再展开核对，标出错频点。
6. **纳入节奏**：对反复错的内容在页边做标记，导入间隔复习或专项练习清单。

## 使用场景与案例

### 场景一：大学专业课随堂

**背景**：板书快、概念密，线性抄满讲义仍不会做题。

**应用**：课上专注右栏，当晚补左栏问题与底部总结；考前以左栏为题库自测。

**结果**：替代低效重读整本讲义，错误模式集中、可针对性补洞。

### 场景二：技术文档与 API 学习

**背景**：参数多、易混，排查问题时翻页成本高。

**应用**：右栏记用法与坑，左栏写「何时用」「与 X 区别」「常见报错对应检查」。

**结果**：先扫线索栏即可定位，缩短从文档到可运行代码的路径。

### 场景三：读书会与非虚构阅读

**背景**：金句多、结构散，读后难以成篇输出。

**应用**：每章若干页康奈尔，底部串联章主旨；左栏形成「问题链」供讨论。

**结果**：写摘要或分享时有现成提纲，观点有出处、有对照。

### 场景四：会议与培训记录

**背景**：决议与待办易淹没在流水账里，会后失忆。

**应用**：右栏记事实与讨论要点，左栏写「谁、何时、交付物、依赖」，总结栏写会议结论。

**结果**：跟进单清晰，减少扯皮与重复对齐成本。

### 场景五：自学考证

**背景**：定义与法条量大，需要记忆与理解兼有。

**应用**：右栏精简条文或公式，左栏编判断题式提示；错题回写总结栏归因。

**结果**：刷题与笔记互相强化，薄弱环节可见、可排期。

## 常见误区

- 左栏只抄标题词没有问题导向，自测强度弱、易变成第二遍摘抄。
- 右栏全文誊写，耗时且不促进加工与取舍。
- 复习从不遮主栏，笔记沦为「漂亮存档」而非提取练习。
- 总结栏空白或仅复制右栏首句，失去概括与迁移训练。
- 页面过满不留白，后续补充、关联与纠错困难。

## 搭配工具

- **SQ3R 阅读法**：Question 阶段产出左栏问题，Review 与康奈尔自测合并。
- **思维导图法**：页内康奈尔承载细节，章末导图串页与串主题。
- **间隔重复法**：左栏问题导入卡片或复习日历，按难度调度。
- **双编码理论**：总结栏旁附微型示意图或流程草图，强化线索。
