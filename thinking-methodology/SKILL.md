# 思维方法论工具箱

## 使用流程

1. **识别问题类型**：根据用户描述，定位所属领域和场景
2. **匹配思维工具**：从下方索引中选择最合适的工具（可多选组合）
3. **读取参考文件**：阅读对应的 `references/` 文件获取详细框架和案例
4. **引导应用**：用工具框架帮用户分析问题、输出结论和行动方案
5. **组合使用**：复杂问题建议组合多个工具，见底部"组合策略"

## 组合策略

复杂问题往往需要多个工具配合使用：

### 常见组合模式

| 场景 | 推荐组合 | 逻辑 |
|------|----------|------|
| 创业规划 | SMART → SWOT → WBS → OKR → PDCA | 定目标→评估→拆解→对齐→迭代 |
| 向上汇报 | 金字塔原理 → PREP → SCQA | 组织→表达→开场 |
| 解决难题 | 5W2H → 5Why → MECE → 鱼骨图 → 六顶思考帽 | 描述→追因→拆分→关联→决策 |
| 自我提升 | 职业锚 → 护城河 → T型人才 → 刻意练习 → GRAI | 方向→定位→路径→执行→复盘 |
| 团队建设 | 塔克曼 → 贝尔宾 → RACI → 情境领导力 → 双因素激励 | 阶段→角色→职责→风格→激励 |
| 情绪管理 | ABC情绪 → 认知重评 → 斯多葛二分法 → 正念冥想 | 觉察→转念→接纳→平静 |
| 学习攻坚 | 费曼 → SQ3R → 康奈尔笔记 → 间隔重复 → 输出倒逼 | 理解→精读→记录→巩固→应用 |
| 时间救急 | 四象限 → 吃青蛙 → 番茄工作 → 批量处理 → 帕金森定律 | 排序→先难→专注→合并→限时 |

### 组合原则

1. **诊断先行**：先用分析工具（5W2H/MECE/5Why）搞清问题
2. **决策居中**：用决策工具（六顶思考帽/SWOT/决策矩阵）选方案
3. **执行收尾**：用管理工具（PDCA/OKR/GTD）落地执行
4. **复盘闭环**：用复盘工具（GRAI/反事实思维）总结提升
