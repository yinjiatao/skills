# Search Strategy Guide

> 针对不同人物类型的高级搜索策略

---

## Type 1: Investor (投资人)

### Characteristics
- 决策驱动型内容（投资决策、市场观点）
- 有公开的投资组合或业绩记录
- 常在金融媒体、播客、会议发言

### Search Queries

#### Core Identity
```
"{name}" investor biography background
"{name}" investment philosophy thesis
"{name}" how I invest strategy
```

#### Decision Patterns
```
"{name}" investment decision rationale
"{name}" why I bought sold thesis
"{name}" mistake loss learned
```

#### Public Content
```
"{name}" letter to shareholders LP
"{name}" podcast interview investment
"{name}" conference speech transcript
```

#### Risk/Reward Profile
```
"{name}" risk management approach
"{name}" concentrated vs diversified
"{name}" time horizon long term
```

### Key Sources
- 13F filings (US institutional investors)
- Investor letters/annual reports
- 播客: Invest Like the Best, My First Million, Bankless
- 会议: Sohn Conference, Milken Institute
- Twitter/X investment threads

### Red Flags
- 过度关注短期交易记录
- 缺乏理念层面的内容
- 只有二手分析，无本人发言

## Type 2: Founder (创始人)

### Characteristics
- 创业历程为核心内容
- 公司成长与个人成长交织
- 决策压力大，反思深度高

### Search Queries

#### Origin Story
```
"{name}" founding story why started
"{name}" origin story interview
"{name}" started company journey
```

#### Growth & Scaling
```
"{name}" scaling challenges hiring
"{name}" company culture building
"{name}" leadership lessons learned
```

#### Failure & Pivot
```
"{name}" failed pivot mistake
"{name}" almost quit shutdown
"{name}" setback adversity
```

#### Decision Points
```
"{name}" why decided to sell IPO
"{name}" co-founder conflict
"{name}" board relationship
```

### Key Sources
- 公司博客/Medium publication
- Y Combinator interviews
- 播客: How I Built This, The Tim Ferriss Show, Lex Fridman
- 书籍/自传
- Twitter/X build-in-public threads
- TechCrunch, The Information profiles

### Timeline Focus
1. **Pre-founding**: 什么促使创业
2. **0-10 employees**: 早期决策模式
3. **Scaling**: 如何应对快速增长
4. **Crisis moments**: 公司生死关头
5. **Exit/transition**: 退出或转型思考

## Type 3: Creator (创作者)

### Characteristics
- 内容产出是主要身份
- 表达风格本身就是维度
- 与受众关系是核心

### Search Queries

#### Creative Process
```
"{name}" creative process workflow
"{name}" how I create write make
"{name}" inspiration source
```

#### Audience Relationship
```
"{name}" relationship with fans audience
"{name}" why I create purpose
"{name}" creative block struggle
```

#### Evolution
```
"{name}" how style changed evolved
"{name}" early work vs now
"{name}" influences mentors
```

#### Business of Creating
```
"{name}" monetization creative business
"{name}" independence vs commercial
"{name}" creative freedom compromise
```

### Key Sources
- 个人网站/博客（最高优先级）
- Newsletter archive (Substack, ConvertKit)
- YouTube videos（观察表达方式）
- Twitter/X threads
- Patreon/创作者平台内容
- 播客访谈（通常是深度内容）

### Content Analysis Focus
- **表达风格**: 幽默/严肃、直白/隐喻、短句/长句
- **主题偏好**: 反复出现的主题和执念
- **互动模式**: 如何回应评论和反馈
- **创作节奏**: 高产期vs低谷期，触发因素

## Source Reliability Ranking

### Tier 1 (Highest)
- 本人撰写的长文/博客
- 深度访谈（1小时以上）
- 播客访谈（无稿对话）
- 本人发布的视频/演讲

### Tier 2 (High)
- 本人授权的书/自传
- 长期合作的专栏文章
- 社交媒体详细帖子

### Tier 3 (Medium)
- 第三方媒体采访
- 会议演讲（可能有准备）
- 员工/同事的描述

### Tier 4 (Use with Caution)
- 简短引用
- 匿名消息
- 竞争对手评价
- 公关稿

---

## Language Considerations

### Multilingual Search
- 搜索人物母语的原始内容
- 注意翻译带来的信息损失
- 文化背景影响表达方式

### Regional Platforms
- 中国: 知乎、微信公众号、微博
- 日本: note.mu、Hatena Blog
- 欧洲: Medium (多语言)
- 全球: Twitter/X, LinkedIn

---

> **Pro Tip**: 最好的搜索策略是组合使用——通过时间线建立框架，通过决策故事填充内容，通过价值观查询验证维度。
