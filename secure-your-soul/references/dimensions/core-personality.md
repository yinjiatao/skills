# MBTI 核心人格维度

MBTI（Myers-Briggs Type Indicator）基于荣格的心理类型理论，从四个维度描述人格偏好。

## 1. 能量取向 (Energy) - E/I

**外向 (Extraversion, E)**
- 从外部世界和人际互动中获取能量
- 喜欢表达想法，通过讨论来思考
- 倾向于行动先于思考
- 喜欢社交活动，兴趣广泛多样

**内向 (Introversion, I)**
- 从内心世界和独处中获取能量
- 倾向于先思考再表达
- 喜欢深度交流，偏好小圈子
- 需要独处时间来恢复精力

## 2. 感知方式 (Perception) - S/N

**实感 (Sensing, S)**
- 关注具体、可观察的事实和细节
- 相信经验，注重现实和当下
- 喜欢实际、具体的信息
- 擅长执行和动手操作

**直觉 (Intuition, N)**
- 关注模式、可能性和整体图景
- 重视想象力和未来可能性
- 喜欢抽象理论和创新想法
- 擅长联想和概念化思考

## 3. 决策方式 (Decision) - T/F

**思考 (Thinking, T)**
- 基于逻辑和客观分析做决策
- 追求公平、真理和效率
- 倾向于批评性问题分析
- 重视能力和胜任

**情感 (Feeling, F)**
- 基于价值观和人际和谐做决策
- 追求共情、理解和和谐
- 考虑决策对他人的影响
- 重视被认可和肯定

## 4. 生活方式 (Lifestyle) - J/P

**判断 (Judging, J)**
- 喜欢有计划、有条理的生活方式
- 追求确定性和控制感
- 喜欢早做决定、按时完成任务
- 重视组织和结构

**感知 (Perceiving, P)**
- 喜欢灵活、开放的生活方式
- 适应变化，保持选择开放
- 倾向于最后期限前完成
- 重视自发性和探索
